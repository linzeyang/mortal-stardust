# Moral Stardust

## 🚀 快速开始

### 1. 安装依赖
```bash
# 运行自动安装脚本
./setup.sh

# 或手动安装
npm install
cd backend && pip install -r requirements.txt
```

### 2. 配置环境
```bash  
# 编辑前端环境变量
vim .env

# 编辑后端环境变量  
vim backend/.env
```

### 3. 启动服务
```bash
# 启动前端 (新终端)
npm run dev

# 启动后端 (新终端)
cd backend && python run.py
```

### 4. 访问应用
- 前端: http://localhost:3000
- 后端API: http://localhost:8000
- API文档: http://localhost:8000/docs
